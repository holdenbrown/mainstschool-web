🔧 How to Test the CMS

  1. Start the Proxy Server: Open terminal and run: netlify-cms-proxy-server
  2. Start Live Server: Right-click on index.html → "Open with Live Server"
  3. Access CMS: Go to http://localhost:5500/admin/
  4. Test Features: Try adding staff members and blog posts
